从array-like or iterable object创建一个新的数组，等价于...运算符

  • Syntax
  • from Array
  • from string
  • fromm a Map
  • from an Array-like object (arguments)
  • 使用箭头函数，相当于map方法

Array.prototype.copyWithin()

在当前数组内部，将指定位置的成员复制到其他位置（会覆盖原有成员），然后返回当前数组。使用这个方法，会修改当前数组。

  • Syntax

target: 序列将被复制到的目标位置
start: 被复制序列的起点，默认为0
end: 被复制序列的终点，默认为arr.length
