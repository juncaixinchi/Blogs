Btrfs文件系统

Btrfs是新一代的Linux文件系统，目标是取代Linux目前的ext3文件系统，改善ext3的限制，特别是单一文件大小的限制，总文件系统大小限制以及加入文件校验和特性。

基本的操作系统命令(一般都需要sudo权限)

  • 安装Btrfs工具
  • 创建btrfs文件系统(以sdb为例)
  • 搜索brtfs文件系统
  • 挂在文件系统
  • 快照（Snapshots）和建立子卷（subvolumes）
  • 挂载子卷
  • 打印所有的文件系统metadata
