My First Javascript Package in Npm

写了个基于node的操作NTFS Alternate Data Stream的库，发布在了npm，特此记录一下流程

创建github项目

在github上新建项目 fs-ads

码好代码，写好测试。

注册npm账号

在本地代码池使用npm adduser创建新的账户

如果已经有账户，可以用npm login来登录账户

发布与更新

使用npm publish发布package

更新package可以使用 npm version <update_type>命令，

其中<update_type>可以是patch, minor, or major，分别对应三个版本号。

npm 会自动提交一个新的commit，新建tag

再次使用npm publish即可更新原有版本
