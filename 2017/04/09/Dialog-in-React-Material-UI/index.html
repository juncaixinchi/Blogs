自己写的dialog

  • 使用ReactTransitionGroup来处理动画，主要是组件进入和退出的动画

  • 使用display: 'flex', alignItems: 'center', justifyContent: 'center'实现居中

  • 基本结构：

    • root: 根div，一个撑满页面的容器，包含transformItem和overlay

    • transformItem：居中显示的包含内容的对话框窗口

    • overlay: dialog后面灰度背景遮罩
